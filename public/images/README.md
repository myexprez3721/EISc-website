# Images Directory

Place your logo here:

## Logo File

**File name**: `eis-logo.png`
**Dimensions**: 200x200px (minimum)
**Format**: PNG with transparent background

The logo is used in:
- Navigation header (50x50px)
- Footer (60x60px)
- Social media links (optional)

Once you upload the logo image here, restart the development server (`npm run dev`) for changes to take effect.
